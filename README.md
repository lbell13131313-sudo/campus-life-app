It is my campus life website, it is intended to mimic how a normal campus website would function.

I used HTML, Javascript, CSS, and Weatherstack api to put this project together.

If the deployment doesn't work the first time around, inspect element, go to console, then click the cors-anywhere link, and then press the grey button. Try it again and the api deployment will work as intended.

The website has many buttons you can press. The one button on all the pages is the home button which will take you back to index.html. There is also the events button which will take you to events.html and the dining button which will take you to dining.html. There is also a button on events.html that when you press it will tell you the weather today so you can be prepared appropriately for any events.

Some stretch goals of mine would be to make the website more sightly. I think it looks good but could look more professional. I also would impliment the api better rather than it just say the weather for today I'd want it to say the weather for the week or the weather on the day of each event. I'd also want a meet the staff page for new students who wanna get to know the teachers. The last thing I'd add is an api menu in dining.html so students can see what food there is.
